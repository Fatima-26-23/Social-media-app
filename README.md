# Social Media App (C++)

This project demonstrates OOP concepts by simulating a simple social media app.
It includes:
- User login
- Posting messages
- Viewing posts
- Basic interactions

## How to run
1. Compile the code using a C++ compiler:
   `g++ social media app.cpp -o social`
2. Run:
   `./social`

## Features
- Uses classes and objects
- Simple menu interface
- Demonstrates inheritance and encapsulation
